Class File_Passwd_Common

Description
Description | Descendents | Vars (details) | Methods (details)

Baseclass for File_Passwd_* classes.

Provides basic operations:

  • opening & closing
  • locking & unlocking
  • loading & saving
  • check if user exist
  • delete a certain user
  • list users
